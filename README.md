# Clean PHP Smarty Blog

Простой блог на чистом PHP, Smarty и MySQL.

## Деплой и запуск

Запустить контейнеры:

```bash
docker compose up -d --build
```

Создать таблицы:

```bash
docker compose exec -T mysql mysql -ublog -pblog_password blog < database/schema.sql
```

Заполнить тестовые данные:

```bash
docker compose exec php php database/seed.php
```

Открыть сайт:

```text
http://localhost:8080
```

## Страницы

- `/` - главная страница с категориями и тремя последними статьями в каждой категории.
- `/category?id=1` - страница категории со списком статей.
- `/category?id=1&sort=views` - сортировка статей по просмотрам.
- `/category?id=1&sort=date` - сортировка статей по дате публикации.
- `/post?id=1` - страница статьи с похожими статьями.
- `/db-check` - простая проверка подключения к базе данных.

## Что реализовано

- Категории и статьи хранятся в MySQL.
- Статья может быть связана с одной или несколькими категориями.
- Главная страница выводит только категории, в которых есть статьи.
- На странице категории есть сортировка и пагинация.
- На странице статьи увеличивается счетчик просмотров.
- Для статьи выводятся 3 похожие статьи из общих категорий.
- Есть скрипт сидинга категорий и статей.
